# 👤 User Management System

A sleek, modern, and fully functional User Management dashboard built with **Tailwind CSS** and **Vanilla JavaScript**. This project features a responsive design, dynamic data handling, and a beautiful dark mode experience.

---

## ✨ Features

- **🚀 Dynamic Registration:** Add users instantly without page refreshes.
- **🌓 Dark Mode:** Fully integrated dark/light mode toggle with system preference detection.
- **📱 Responsive Design:** Optimized for mobile, tablet, and desktop views.
- **♻️ Smart Table Management:** 
  - Automatic row re-indexing upon removal.
  - Interactive "Remove" actions.
  - Live user count badge.
- **🔔 User Feedback:** Animated success messages after every submission.
- **🎨 Modern UI:** Built with the Slate & Indigo color palette for a professional look.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **Tailwind CSS:** Modern styling and dark mode utility classes.
- **JavaScript (ES6+):** DOM manipulation and state management.
- **Lucide/Heroicons:** SVG iconography for the theme toggle.
